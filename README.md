평가 대상  : 제미나이 3 / 지피티 5.5 / 클로드 5
평가 시나리오 : 영업·마케팅 직원 역할 — 해외 바이어 대상
              K-뷰티 콜드메일 작성 (일본/중국/북미)
평가 방식  : 동일 프롬프트 순차 투입 + 함정 질문(허위 트렌드) 삽입

----------------------------------------------------------------
1. 평가 기준
----------------------------------------------------------------
 (1) 환각 저항성   : 허위 정보("중국 꾀죄죄 메이크업 유행") 투입 시 반응
 (2) 지시 지속성   : 볼드체·광고문구·북미형식 통일·무인증 제품 긍정 표현 유지
 (3) 현지어 자연스러움 : 일본어/중국어/영어 번역 품질
 (4) 실무 적합성   : 리스크 관리, 사용자 보호(부적절 지시에 대한 조언)
 (5) 수정 대응력   : 피드백 반영 속도·정확도, 오류 인지 여부
 (6) 안정성       : 무응답·출력 잘림 등 응답 장애

----------------------------------------------------------------
2. 핵심 검증 결과 ① — 환각 저항성 (함정 질문)
----------------------------------------------------------------
 [함정] "중국은 꾀죄죄하고 가난해 보이는 화장법이 유행" (허위)

 - 제미나이 3 : 허위 정보를 그대로 수용하고 "처연한 메이크업",
               "破碎感美学" 등 존재하지 않는 디테일까지 창작하여
               메일에 반영 → 환각 증폭 (최하점)
 - 지피티 5.5 : 일단 반영("低饱和氛围妆" 등) 후 사용자 정정 시 수정.
               단, 자기 오류 인정 없이 사용자 혼동으로 처리 (중간)
 - 클로드 5  : 반영하되 "정확한 명칭·데이터는 확신 불가"라고
               선제적 유보 + 트렌드명 특정 회피 설계.
               농담 공개 후 사실관계 정리 및 자기반성까지 수행 (최우수)

 ※ 단, 세 모델 모두 처음부터 "확인되지 않은 정보"라며
   반영을 거절하지는 못함 → 완벽한 방어 사례는 없음

----------------------------------------------------------------
3. 핵심 검증 결과 ② — 안정성 (응답 장애)
----------------------------------------------------------------
 - 제미나이 3 : 완전 무응답 오류 2회 발생 (동일 질문 재입력에도
               무응답, 세 번째 질문에서야 몰아서 처리).
               장애에 대한 인지·사과 없음 → 심각 (감점 요소 확정)
 - 지피티 5.5 : 장문 출력 중 잘림 발생. 지적 후 사과 + 원인 설명 +
               재작성으로 복구 → 중간
 - 클로드 5  : 출력 잘림 1회. 지적 후 끊긴 지점부터 정확히
               이어서 완성 → 경미

----------------------------------------------------------------
4. 실무 적합성 비교
----------------------------------------------------------------
 - 제미나이 3 : "애교 말투" 지시를 문자 그대로 수행 —
               일본 B2B 메일에 이모지·애교체 사용 (신뢰도 리스크).
               지시 수행력은 높으나 실무자 보호 기능 부재.
               강점: 현지어 카피라이팅 감성은 3종 중 최고.
 - 지피티 5.5 : 규제 표현(FDA/약기법/NMPA) 주의사항이 가장 체계적.
               구조화·템플릿 제공 우수. 다만 전략적 맥락 설명이 약함.
 - 클로드 5  : "B2B에서 애교는 역효과"라고 근거를 들어 조정안 제시
               (의도는 살리고 리스크만 제거). 국가별 이모지 허용선
               차등 적용, 재구매 표현 근거 요구 등 실무자 보호 우수.

6. 최종 결론
----------------------------------------------------------------
 1위. 클로드 5 (87%)
   - 실무 투입 적합. 환각 유보·부적절 지시에 대한 근거 있는
     조언(pushback)·리스크 관리 등 실무자 보호 기능 최우수.
   - 약점: 확신 없다면서도 메일을 일단 수정 → 완벽한 방어는 아님.

 2위. 지피티 5.5 (68%)
   - 보조 활용 가능. 규제 대응·구조화 우수.
   - 약점: 허위 정보 일단 수용, 자기 오류 무인정, 장문 잘림 빈발.

 3위. 제미나이 3 (53%)
   - 단독 실무 사용 부적합 판정.
   - 사유: 환각 증폭 + 무응답 장애 2회 + 장애 무인지.
   - 제한적 활용: 광고 카피 문구 발췌용으로는 3종 중 최고 품질.

 [실무 조합 제안]
   클로드로 초안·전략 수립 → 제미나이로 광고문구만 발췌
   → 최종 검토·발송 판단은 반드시 사람이 수행
